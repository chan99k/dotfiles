# Reference Videos

Curated list of high-performing videos by genre. Auto-loaded in Phase 0 and passed to the Success Analyst (Phase 1) for pattern extraction.

## How to Use

1. Add entries to the relevant genre table below.
2. Mark top performers with ★ in the Star column.
3. The Success Analyst will prioritize ★ entries for deep analysis.
4. Keep the list focused — 5-10 per genre is ideal. Remove underperformers periodically.

**Fields**:
- **Title**: Video title (Korean or English)
- **URL**: Full YouTube URL
- **Views**: Approximate view count at time of recording (e.g., 120K, 1.2M)
- **Key Factor**: One-line reason this video performed well
- **Star**: ★ for top reference, blank otherwise

**How to find videos**: Search each channel's "인기 동영상" tab on YouTube, sort by views, pick 2024-2025 uploads.
